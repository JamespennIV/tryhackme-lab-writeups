# TryHackMe Lab Writeups

## Overview

This repository documents TryHackMe cybersecurity labs I've completed, written up in a portfolio-safe format. The goal is to show my learning progress, technical methodology, tools I used, and the key concepts I walked away with — without exposing flags, direct answers, or anything that crosses into unauthorized walkthrough territory.

These writeups cover cybersecurity operations, threat analysis, investigation, vulnerability management, and technical documentation.
